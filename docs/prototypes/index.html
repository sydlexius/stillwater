Stillwater UX proposal

Each link opens a single proposed screen at full viewport — no canvas, no zoom, no side-by-side overlays. Use your browser's back button to return here. The current Stillwater UI for each screen is on a separate page for direct comparison.

⌘K / Ctrl+K opens the command palette from any proposed screen — jump between screens, search settings, run actions. Or press g then a letter (dashboard, artists, reports, findings, settings, onboarding) to jump directly.
